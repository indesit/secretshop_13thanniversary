# Decisions

## 2026-04-25 - Campaign format

Запускаємо кампанію у форматі **Birthday Week Light / MVP**, а не чекаємо повного продакшн-пакета.

Причина:

- день народження вже настав;
- є подарунки, смаколики, кульки і один key visual;
- немає повного комплекту банерів / роликів / лендингу;
- відкладати комунікацію недоцільно.

## 2026-04-25 - Main message

Затверджена коротка формула:

> Secret Shop святкує 13 років.  
> Протягом святкового тижня даруємо подарунки до покупок.  
> Чекаємо вас у магазинах та онлайн.

## 2026-04-25 - Discount policy

Знижка не є центральною механікою кампанії.

Фокус:

- подарунки;
- смаколики;
- святкова атмосфера;
- подяка клієнтам;
- м’який привід для покупки.

## 2026-04-25 - Visual production principle

На старті використовується один key visual як master-основа для всіх каналів.

Пізніше можна зробити адаптації:

- desktop banner;
- mobile banner;
- email header;
- stories;
- post;
- popup.

## 2026-04-25 - Operational rule

Перед будь-якою розсилкою мають бути зрозумілі 4 речі:

1. Кому пишемо.
2. Що пропонуємо.
3. Куди ведемо.
4. Яку дію очікуємо.

Для MVP-відправки:

- аудиторія: база клієнтів Secret Shop;
- пропозиція: подарунки до покупок протягом святкового тижня;
- напрямок: магазини та онлайн;
- дія: прийти / перейти / купити протягом тижня.
