# Assets

Сюди складати всі візуальні матеріали кампанії Secret Shop 13th Anniversary.

## Рекомендована структура

```text
assets/
├── key-visual/       # Головний master visual
├── website/          # Desktop / mobile банери
├── email/            # Email header / campaign preview
├── stories/          # Instagram stories 1080x1920
├── posts/            # Instagram posts 1:1 або 4:5
├── store/            # Друковані матеріали для магазинів
└── screens/          # Ролики / екрани, якщо будуть
```

## Поточний MVP

На старті є один key visual. Його можна використовувати як master-основу для всіх каналів до появи окремих адаптацій.

## Naming convention

Рекомендовано називати файли так:

- `secretshop-13th-key-visual-desktop-v01.png`
- `secretshop-13th-key-visual-mobile-v01.png`
- `secretshop-13th-email-header-v01.png`
- `secretshop-13th-story-01-v01.png`
- `secretshop-13th-post-v01.png`

## Мінімальний набір адаптацій

Must-have:

- website desktop banner;
- website mobile banner;
- email header;
- 1-3 stories.

Nice-to-have:

- Instagram post;
- popup visual;
- screen loop;
- in-store print banner.
